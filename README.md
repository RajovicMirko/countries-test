# countries-test

Test application for countries overview

## Tech stack

- React js

## Get Project

### Clone repositorie - terminal

```
git clone https://github.com/RajovicMirko/countries-test.git
```

### Enter folder - terminal

```
cd .\countries-test\
```

### Init dependencies - terminal

```
npm i
```

### Start project - terminal

```
npm start
```
